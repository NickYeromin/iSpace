# 🌞 iSpace
## 🚀About the Project
The idea of the project is to create an interactive application for viewing the latest news, blogs, and reports about space.
All data is fetched via API, and the user interface is built with React + Vite, ensuring speed and a modern application architecture.

The goal of the project is to demonstrate working with APIs, dynamic content, and modern frontend development tools.
